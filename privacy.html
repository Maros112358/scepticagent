Privacy Policy

ScepticAgent Chrome Extension — Last updated: March 2026

Overview

Research Assistant is a Chrome extension that helps you critically analyse webpages using AI. This policy explains what data the extension processes and how it is handled.

Data we do NOT collect

The developer of this extension does not:

  • Collect, store, or transmit any data to their own servers
  • Track usage, behaviour, or analytics of any kind
  • Have access to your API keys, page content, or conversations

Data sent to third-party AI providers

When you use Research Assistant, the text content of the webpage you are viewing and any questions you type are sent directly from your browser to the AI provider you have configured (Anthropic, OpenAI, or Google). This is the core function of the extension.

What is sent:

  • Page content — the visible text of the webpage you are analysing
  • Chat messages — questions you type in the Chat tab
  • Your API key — used to authenticate each request; it is never sent anywhere other than directly to the respective provider's API endpoint

These transmissions are governed by the privacy policies of each provider:

  • Anthropic Privacy Policy
  • OpenAI Privacy Policy
  • Google Privacy Policy

API key storage

Your API keys are stored exclusively in your browser's local extension storage (chrome.storage.local) on your own device. They are never synced to the cloud, shared with the extension developer, or sent to any server other than the respective AI provider's API endpoint.

Permissions used

  • activeTab / scripting — to read the text content of the page you are analysing
  • storage — to save your API keys and settings locally on your device
  • sidePanel — to display the extension's interface in Chrome's side panel
  • tabs — to open links (e.g. the API key console) in new tabs

Children's privacy

This extension is not directed at children under the age of 13 and does not knowingly collect information from children.

Changes to this policy

If this policy is updated, the "Last updated" date at the top of this page will be revised. Continued use of the extension after changes constitutes acceptance of the updated policy.

Contact

If you have questions about this privacy policy, please open an issue on the extension's support page.
